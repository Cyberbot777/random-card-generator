# Random Card Generator

A simple vanilla JavaScript project that displays a randomly generated playing card each time the page loads.

## Features

- Random suit (♠ ♥ ♦ ♣)
- Random card value (A, 2–10, J, Q, K)
- Responsive design
- Built with Vite, HTML, CSS, and JavaScript

## Getting Started

```bash
npm install
npm run dev
